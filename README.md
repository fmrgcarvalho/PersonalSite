## ✨ Features

- Dark mode premium design
- Single-page with smooth scroll navigation
- Animated skill bars (triggered on scroll)
- Number counter animations
- Scroll reveal animations
- Expandable certifications list (20+)
- Fully responsive (mobile-first)
- Self-contained — no external dependencies except Google Fonts

## 🛠 Tech Stack

Pure HTML5 · CSS3 · Vanilla JavaScript · Google Fonts (DM Serif Display + Geist)
